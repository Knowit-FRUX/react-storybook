# Demo of React with Storybook

This app was created from a boilerplate setup. 

<pre>npx create-react-app react-storybook</pre>
<pre>cd react-storybook</pre>
<pre>npx -p @storybook/cli sb init</pre>
<pre>npm run storybook</pre>
